# bodybuddy-calculator
바디버디 코딩테스트 (계산기 구현)

<br />

## 기능
- 첫 자릿수 연산자 제한
- 연산자 연속 입력 제한
- 숫자 최대 15자리까지 입력 제한
- 음수 처리
- 계산 결과 DB 저장 (10번 요청 시 Error 후 다시 재요청)
- 제목 클릭 시, 저장된 모든 결과 alert로 표현

<br />

## 실행 방법
<br />

백엔드
```
cd back
node index.js
```

프론트엔드
```
cd front
npm start
```
